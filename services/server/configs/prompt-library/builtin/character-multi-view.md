用当前选择的图片，生成角色设定多视图：

1. 以参考图中的角色为唯一主体，保持角色身份、年龄感、发型、脸型、服装轮廓、关键道具和核心气质一致。
2. 输出一张白底角色设定合成大图，包含正面全身、侧面全身、背面全身、面部特写、配色方案、服装/道具局部细节和比例尺。
3. 多个视图之间比例统一、姿态自然、结构清晰，角色不要换装、不要增加新角色、不要改变物种或时代设定。
4. 画面像专业动画/游戏角色设定稿，排版整洁，留白充足，细节标注以简洁视觉标签呈现。
5. 避免复杂背景、环境叙事、强透视、夸张表情和遮挡主体的装饰。
