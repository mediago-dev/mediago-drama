# jianyingdraft

Go library generated by One CLI. Layout follows
[golang-standards/project-layout](https://github.com/golang-standards/project-layout).

## Quick start

```bash
go mod tidy
task check
task test
```

## Layout

```text
pkg/greeter/           # public API — what consumers import
go.mod                 # module declaration
Taskfile.yml           # go-task tasks (fmt / vet / test / tidy / check)
LICENSE                # Apache-2.0
```

The starter ships with a single placeholder package `pkg/greeter/`. See
`CLAUDE.md` for the full set of directories from
`golang-standards/project-layout` and when to add each.

## Use as a dependency

```bash
go get github.com/example/jianyingdraft
```

```go
import "github.com/example/jianyingdraft/pkg/greeter"

func main() {
    fmt.Println(greeter.Greet("world"))
}
```
