# scene.v1｜场景设定文档结构

你正在生成场景设定文档。

输出规则：
- 直接输出最终 Markdown，不解释模板。
- 根据剧情自动提取关键场景。
- 每个场景使用二级标题。
- 场景数量由剧情需要自动决定。
- 场景内部内容由 AI 根据场景特点自行组织。
- 每个场景必须包含可直接用于生图的 Prompt。
- 场景 Prompt 必须强调无人、纯场景、无字幕、无水印、无 UI。
- 不输出场景提取清单。
- 不输出空字段、占位符或“待补充”。
