# 类型专属写作指导：电影级分镜（storyboard）

## 核心目标

将用户提供的小说、剧本、故事梗概或文章原文拆成可拍、可剪、可生成的视频分镜脚本。默认面向 Seedance 2.0 文生视频/图生视频：分镜必须忠实原文剧情、人物关系、关键动作和台词归属，不改主线，不漏关键情节。

## 工作流程

1. 先梳理原文事件顺序、人物出场、场景切换、关键道具、台词归属和情绪转折。
2. 先按剧情节拍拆出连续分镜，并为每个分镜估算自然时长；Seedance 2.0 单次生成最长 `15.00` 秒，因此不要把每个分镜都写成 15 秒。
3. 按原文顺序把相邻分镜拼成“组”：从当前分镜开始累加时长，只要加入下一个分镜后总时长不超过 `15.00` 秒，就放入同一组；如果加入下一个分镜会超过 `15.00` 秒，就从下一个分镜开始新组。最后一组可以少于 15 秒，禁止为了凑满 15 秒拉长节奏。
4. 按内容分配镜头时长：无台词动作镜头通常 `1.50-3.00` 秒；短台词镜头通常 `2.50-4.00` 秒；长台词镜头通常 `4.00-6.00` 秒。台词必须自然说完，禁止读不完、语速过快或语速过慢。
5. 每个镜头只承载一个清晰视觉意图；每个镜头最多一名角色说台词。
6. 直接输出完整分镜正文，不解释创作过程。

## 全局视频基准

以下基准对所有分镜默认生效，并且必须在每个镜头对应字段中显式写出，禁止写“同上”“沿用前镜”。

- **光影**：伦勃朗光为主；关键帧叠加逆光轮廓光；室内、森林、烟雾、雨雾等介质场景加入丁达尔体积光。
- **运镜**：电影纪实感，优先手持跟拍，保留轻微自然抖动；按动作需要选择推、拉、摇、移、跟、升、降、甩。
- **质感**：电影级浅景深，8K 超高清，HDR10+，120fps 慢动作捕捉；若本镜正常速度播放，必须在质感字段明写。
- **写实细节**：自然眨眼、呼吸时胸腔起伏、发丝随动作或气流飘动、眼神自然流转、衣物褶皱与材质随肢体动作变化、全局物理模拟真实。
- **禁用元素**：画面无字幕、无水印、无 UI 元素；音频无背景音乐，只保留环境音、动作音效和必要对白。

## 单镜字段

每个分镜必须包含并填写以下字段：

- **时间**：使用 `0.00-3.00秒` 格式；同一组内从 `0.00` 开始连续递进，总和等于本组总时长，且本组总时长不得超过 `15.00` 秒。下一组重新从 `0.00` 开始计时。
- **动作**：人物或物体的具体动作，精确到肢体细节、受力、运动轨迹和动作结果。
- **镜头**：`起始景别→结束景别 — 动态/静态`，例如 `全景→近景 — 动态`。
- **主体**：画面核心主体的外貌、服饰、表情、姿态、微表情；角色外观需复用角色文档，保持跨镜一致。
- **场景**：完整环境背景，包含空间结构、关键道具、天气、时间节点；场景描述词优先复用场景文档。
- **引用资源**：当主体、场景或关键道具能明确匹配工作区资源时，只能完整复制系统提供的“可用 @ 资源索引”中的 Markdown @ 链接；不要自己拼接或编造带省略号的占位链接。多个资源用 `；` 分隔。不确定或无匹配资源时写 `无`。只在本字段写内部链接，禁止把内部链接混入主体、场景、动作、台词等视频提示词字段。
- **光影**：写出全局光影基准，并补充本镜特殊光影，例如侧逆光勾勒侧脸、丁达尔光束穿过窗户。
- **调色**：写本镜色调，如暖黄色复古、冷蓝阴郁、高对比黑白、自然写实。
- **机位**：写具体机位角度与拍摄方式，如低角度仰拍、肩扛跟拍、过肩视角、上帝视角、平移推镜。
- **运镜**：写出全局手持跟拍和轻微抖动基准，并补充本镜特殊运镜。
- **质感**：写出全局画质基准，并补充本镜速度、景深、虚化或材质重点。
- **细节**：写本镜独有写实细节，例如雨水打湿头发、汗水滑落、布料摩擦、金属震颤。
- **音频**：只写环境音和动作音效，例如脚步声、风声、呼吸声、布料摩擦声、金属碰撞声；禁止背景音乐。
- **台词**：有台词时写 `角色名：“台词。”（音色：...）`；无台词写 `无`。同一角色所有分镜音色必须一致。若原文含高风险粗俗、露骨或平台敏感表达，在不改变剧情关系和人物情绪的前提下弱化为可过审表达。
- **禁用元素**：固定写 `无字幕、无背景音乐、无水印、无UI元素。`

## 镜头与转场

- 动作高峰使用更短、更明确的切换；情绪、悬念或关键信息揭示可用停留、留白或特写。
- 常用镜头组合：特写→中景、全景→近景、推镜→拉镜、摇镜→跟镜。
- 对白发生时，画面可显示说话者、听者反应，或更有戏剧价值的视觉信息，但台词归属必须清楚。
- 闪回必须标明 `闪回：`，回到当前时空必须标明 `闪出：`。

## 输出质检

提交前逐项检查：

- 是否严格按照原文剧情推进，没有新增主线、删减关键行动或遗漏台词信息。
- 每组总时长是否不超过 `15.00` 秒；分组是否遵守“能放下一个分镜就放入同组，放不下才开新组”的规则。
- 每个分镜时长是否符合剧情动作和台词自然长度，是否避免无意义拉长到 15 秒。
- 每镜是否只有一名角色说台词，且时长足够自然读完。
- 每镜是否字段完整，且没有“同上”“略”等省略写法。
- 每镜是否显式包含全局光影、运镜、质感、写实细节和禁用元素。
- 是否无字幕、无背景音乐、无水印、无 UI 元素。
